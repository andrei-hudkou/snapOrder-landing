# Production SOP (Fast, Mobile-First)

Use this before uploading creatives to Meta Ads Manager.

## 1) Required Dimensions

- Static feed creative: `1080x1350` (4:5)
- Vertical video creative: `1080x1920` (9:16)
- Video duration: 12-20 seconds

## 2) Export Settings

- Static: JPG, high quality, target under 1.5 MB
- Video: MP4 (H.264), 24fps or 30fps, target under 30 MB
- File naming: use fixed IDs (`C1_...` to `C6_...`)

## 3) Safe Zones

- Vertical video (`1080x1920`):
  - keep critical text in center area
  - avoid top and bottom UI overlap zones
- Static (`1080x1350`):
  - keep headline and CTA away from extreme top/bottom edges

## 4) Plain-Copy Rules

- One clear promise per creative
- No buzzwords, no abstract claims
- Prefer short concrete lines
- Use CTA text exactly: `Get early access`

## 5) QA Pass (must pass before publish)

- [ ] Hook visible in first 1-2 seconds (video) or instantly (static)
- [ ] Text readable on mobile without zoom
- [ ] Message matches one angle: pain, time, or control
- [ ] Brand accents use only `#6941c6`, `#53389e`, `#1d2939`
- [ ] Creative ID in filename matches tracker and ad name
- [ ] Link opens and form submits on phone
