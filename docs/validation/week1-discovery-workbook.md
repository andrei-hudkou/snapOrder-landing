# Week 1 Discovery Workbook

## Week Objective
Find the strongest ICP based on observable demand signals and competitor gap analysis (without interviews).

## Hypotheses to Test
1. Sellers managing orders in DMs lose orders due to fragmented chat records.
2. Screenshot-based intake is easier than manual CRM entry.
3. Sellers will accept AI-drafted order records if correction is fast.

## No-Interview Validation Tasks (Week 1)
1. Run competitor/substitute mapping across direct tools and adjacent alternatives.
2. Build one landing page with 2-3 value proposition variants.
3. Run low-budget traffic tests (search/social/community outreach).
4. Track qualified signup behavior and intent (not vanity clicks).
5. Collect asynchronous intent data via form questions and optional waitlist survey.

## Competitor Research Sprint (Parallel, 3 hours)
- Build list of direct tools and substitutes.
- Capture positioning, onboarding friction, and pricing cues.
- Extract "why users still use spreadsheets/notes."

## Scoring Rubric for ICP Selection
- Qualified conversion rate by segment (35%)
- Cost per qualified signup (25%)
- High-intent action rate (20%)
- Message clarity and resonance (10%)
- Estimated willingness to pay from async responses (10%)

## End-of-Week Deliverables
1. Ranked ICP table with weighted scores from behavioral data.
2. Top 5 objections and friction points from form/open-text responses.
3. Competitor gap map with strongest differentiation angles.
4. Recommendation: one ICP for Week 2 fake-door optimization.
