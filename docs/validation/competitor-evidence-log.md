## Positioning Gaps Observed

1. No tool focuses on screenshot-first order capture. All competitors assume API integration or manual inbox.
2. Existing options are complex for solo sellers with low process maturity.
3. Room for "capture first, structure later" positioning vs "set up your CRM first."
4. Main risk: users may see this as "another CRM" unless onboarding is radically simple.
5. Pricing gravity: solo sellers pay $0-15/mo for tools. Above $15/mo needs strong justification.

## Open Validation Questions

1. Is screenshot upload a compelling behavior shortcut vs copying text?
2. Which extracted fields are truly required on day one?
3. Is "do not lose orders" stronger than "save admin time" as core message?
4. What monthly price feels "no-brainer" for solo sellers with <100 orders/month?
5. Do sellers already screenshot conversations? How often?

## Confidence Assessment
- Medium on market signal (competitors exist but none own the screenshot angle).
- Low-to-medium on pricing assumptions (need behavioral data).
- High that manual substitutes (notes/sheets) are the true baseline competitor.
